# Styles

**Purpose:** Global and shared CSS files.

**What belongs here:**

- Global stylesheets (typography, colors, resets, etc.)
- CSS utilities and base styles
- Shared style variables

**What does NOT belong here:**

- Component-specific styles (keep those in component files or co-locate with components)
- Feature-specific styles
- Configuration files (those belong in project root)

**Rules:**

- Keep files at global scope only
- Avoid feature or component-specific styling here
